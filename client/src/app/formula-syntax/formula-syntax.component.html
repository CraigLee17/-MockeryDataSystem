Formulas Syntax

Formulas allow you to use Javascript code to generate data based on custom logic.

Operators

+ - * / %

Logic

< > <= >= == != and or

Functions

Mockery data is built on top of faker, chance and casual. You can easily make use of the functions provided by these libraries.

Examples:
Faker

faker.internet.email()

faker.company.companyName()

faker.date.past()

faker.image.imageUrl()

Please go the Faker.js to see more examples.

Chance

chance.address()

chance.age({{ '{' + "type: 'child'" + '}'}})

chance.ipv6()

chance.country()

Please go the Chance.js to see more examples.

Casual

casual.username

casual.rgb_hex

casual.domain

casual.state

Please go the Casual.js to see more examples.
